# mo2-overlayfs

This is a library to provide a similar interface to [USVFS](https://github.com/ModOrganizer2/usvfs) to natively run
ModOrganizer 2 on Linux.

## Requirements

Older versions may also work but are not tested.

- CMake 3.31
- fuse-overlayfs 1.14
- GCC 14
- Kernel 6.13
- Qt Base 6.8
- spdlog 1.15.1
